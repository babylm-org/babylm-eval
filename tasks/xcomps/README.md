# XCOMPs (Cross-lingual COMPositional Skills)

### Paper

Title: `Cross-lingual compositional skills in large language models`

Homepage: https://huggingface.co/datasets/fpadovani/xcomps-dataset

XCOMPs evaluates cross-lingual compositional skills through minimal pairs testing syntactic and semantic understanding across multiple languages.
