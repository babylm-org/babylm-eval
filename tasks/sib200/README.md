# SIB-200

### Paper

Title: `SIB-200: A Large-Scale Text Classification Benchmark for Multilingual Language Models`

Homepage: https://huggingface.co/datasets/Davlan/sib200

SIB-200 is a topic classification benchmark covering 200+ languages with 7 categories: science/technology, travel, politics, sports, health, entertainment, and geography.
